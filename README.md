# EduStream | Advanced Student Management Portal

EduStream is a high-performance web dashboard developed to showcase advanced JavaScript concepts, data structure management, and dynamic UI rendering. This application provides a centralized interface for student administration and academic planning.

## 🚀 Key Modules & Functional Logic

### 1. Student Authentication System
- **Functionality:** Validates user input and generates dynamic greetings.
- **Tech Detail:** Leverages ES6 Function declarations and Template Literals for clean, readable string manipulation.

### 2. Enrollment Laboratory (Array Management)
- **Functionality:** A real-time interface to manipulate student records.
- **Methods Demonstrated:** - `push()`: Adding new students to the end of the registry.
  - `pop()`: Removing the last entry for error correction.
  - `shift()`: Removing the first entry from the queue.
  - `unshift()`: Prioritizing new entries at the start of the list.

### 3. Smart Course Catalog Search
- **Functionality:** Instant search feature to check subject availability.
- **Logic:** Implements the `.indexOf()` algorithm to navigate through data arrays and provide visual status feedback.

### 4. Academic Study Planner
- **Functionality:** A robust task management tool for students to organize their curriculum.
- **Implementation:** Features dynamic DOM creation where tasks are stored in arrays and rendered into the UI using automated loops.

## 🛠️ Technical Stack
- **HTML5:** Semantic architecture for improved accessibility.
- **CSS3:** Custom properties (variables), Grid-based layouts, and fluid responsiveness.
- **JavaScript (ES6):** Advanced DOM manipulation, event-driven programming, and array-based state management.

## 📂 Deployment & Directory Structure
This project is optimized for GitHub Pages. The core files (`index.html`, `style.css`, `script.js`) are located in the root directory for seamless deployment and fast loading times.
