# EduStream | Advanced JavaScript Student Portal

EduStream is an interactive dashboard developed to showcase core and advanced JavaScript concepts. This project serves as a functional lab for testing array methods, dynamic DOM rendering, and user input handling.

## 🚀 Key Modules & Features

### 1. Student Authentication & Greeting
- **Logic:** Uses custom functions and ES6 Template Literals.
- **Feature:** Generates a personalized welcome message for users.

### 2. Enrollment Lab (Array Methods)
- **Logic:** Demonstrates the practical use of `push()`, `pop()`, `shift()`, and `unshift()`.
- **Feature:** Real-time management of a student registry list with instant UI updates and console logging.

### 3. Course Catalog Search
- **Logic:** Implements the `indexOf()` method to search through a data array.
- **Feature:** Provides visual feedback (Success/Error states) based on subject availability.

### 4. Academic Study Planner (Mini Project)
- **Logic:** A complete Todo-style application using array storage and dynamic list rendering.
- **Feature:** Add tasks, remove the last task, or delete specific items from the planner.

## 🛠️ Technical Stack
- **HTML5:** Structured with a modern CSS Grid-based dashboard layout.
- **CSS3:** Custom styling using CSS Variables, Hover effects, and Responsive design.
- **JavaScript (ES6):** - Functional programming.
  - Array manipulation and searching.
  - Advanced DOM Event Listeners.
  - Conditional rendering.

## 📂 Installation & Usage
1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. Open the **Browser Console (F12)** to view the array operations and logic execution in real-time.
